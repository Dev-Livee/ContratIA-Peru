# Architecture Decision Records

Documentamos aquí las decisiones de arquitectura importantes del proyecto.

## Formato

Cada ADR sigue esta estructura:
- **Estado**: Propuesto | Aceptado | Deprecado
- **Contexto**: El problema o necesidad
- **Decisión**: Qué decidimos y por qué
- **Consecuencias**: Positivas y negativas
